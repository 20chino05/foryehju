IRS & California Tax Resources
Free official resources your parents can use — no attorney needed for some of these.
🇺🇸 Free IRS Programs
🛡️ Taxpayer Advocate Service (TAS)
Free independent help from the IRS. If your case is causing hardship, they can intervene. Call 1-877-777-4778. Korean language available.
💰 Offer in Compromise (OIC)
Settle tax debt for less than owed if you can't pay in full. IRS pre-qualifier tool is free. Best handled with an attorney for your dad's situation.
✂️ Penalty Abatement
First-Time Penalty Abatement (FTA) can remove penalties if it's a first offense. Your mom may qualify. Ask any attorney about this first.
📅 Installment Agreement
Pay over time. You can set this up yourself online for under $50k owed. An attorney helps negotiate terms for larger amounts.
⏸️ Currently Not Collectible (CNC)
IRS pauses collection if you truly can't pay. Buys time. Ask your attorney if your dad qualifies — given unemployment reporting, this may be key.
🏛️ California State Resources
⚖️ CA State Bar Attorney Search
Verify any CA attorney's license status, discipline history, and good standing. Free public database. Always check before hiring.
📞 State Bar Lawyer Referral Service
Certified referral services in LA and OC area. Low-cost consultations ($35–$50 often). Some Korean-speaking attorneys in the network.
🏠 CA Franchise Tax Board — Taxpayer Rights Advocate
Helps with FTB issues. Korean interpreter available by request: 1-800-852-5711.
🇰🇷 Korean Community Resources
🏛️ Korean American Bar Association of LA (KAGLA)
Referrals to Korean-speaking attorneys. Some pro bono slots. Good first call if you need a Korean-speaking tax attorney.
📰 Korea Daily / 한국일보 (Koreadaily.com)
Many Korean tax attorneys and CPAs advertise here. Search "세금 변호사" or "tax attorney" in their classifieds. Real practices, local to LA/OC.
📻 Radio Korea (라디오 코리아)
1580 AM. Korean tax attorneys often advertise here and sometimes do free on-air consultations. Your parents can listen and call in.
🌐 Host This App Free
GitHub Pages gives you a permanent URL (like yourname.github.io/tax-finder) that works on any device, no API key needed in the file — just open the URL.
Option A — GitHub Pages (recommended, free forever)
1. Go to github.com and create a free account if you don't have one.
2. Click the + icon → New repository. Name it tax-finder. Set it to Public. Click Create.
3. On the repo page click uploading an existing file, drag in the downloaded korean-tax-finder.html, then rename it to index.html before committing.
4. Go to Settings → Pages. Under Source select Deploy from a branch → main → / (root). Click Save.
5. Wait ~1 minute. Your app lives at https://yourusername.github.io/tax-finder — share that link with your parents.
✅ Once hosted on GitHub Pages the API key card disappears — the app works automatically because the Anthropic header is allowed from real domains.
Option B — Netlify Drop (fastest, 30 seconds)
1. Go to app.netlify.com/drop
2. Drag and drop the korean-tax-finder.html file onto the page.
3. Netlify instantly gives you a URL like random-name.netlify.app. Done.
⚠️ Free Netlify sites expire after 24 hours unless you create an account. GitHub Pages is permanent.
Option C — iPhone / iPad without hosting
1. Get a free API key from console.anthropic.com
2. AirDrop the HTML file to your iPhone/iPad → open in Safari → tap Share → Add to Home Screen
3. Enter the API key in the yellow card on first open — it saves permanently to that device.
ℹ️ The Verified Attorneys tab always works without any API key or internet — all data is baked in.
